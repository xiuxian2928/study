console.assert()

1) console.assert() 예제 - 표현식이 false인지 확인

<script>
 console.assert(document.getElementById("wmakers"), "id='wmakers' 없음");
</script>

2) console.assert() 정의

표현식이 'false' 일 때만 콘솔에 메시지 작성

3) console.assert() 구문

console.assert(expression, message)

expression : 필수. 참거짓 판명할 표현식.

expression : 필수. 표현식이 거짓인 경우 띄울 메세지. (**문자열, 객체나 배열 등 입력가능)

4) console.assert() 예제 - message 매개변수에 객체 입력

<script>
 let wmakers = {site:"WMakers", domain:"wmakers.com"};
 console.assert(document.getElementById("wmakers"), wmakers);
</script>

5) console.assert() 예제 - message 매개변수에 배열 입력

<script>
 let wmakers = ["WMakers", "wmakers.com"];
 console.assert(document.getElementById("wmakers"), wmakers);
</script>

console.clear()

1) console.clear() 예제 - 콘솔창 정리

<button>Console 지우기</button>

<script>
 function clearConsole(){
  console.clear();
 }
</script>

2) console.clear() 구문

console.clear()

반환값 : 콘솔 청소 후, 'Console was cleared' 메세지 띄움

console.count()

1) console.count() 예제 - 콘솔창에서 숫자 세기

<script>
 for(i = 0; i < 10; i++){
  console.count();
 }
</script>

결과값:
default: 1
default: 2
...
default: 10

2) console.count() 정의

호출된 횟수를 콘솔에 표기

- 기본 label은 'default' 이며, 별도 지정 가능

3) console.count() 구문

console.count(label)

label : 선택. 호출 이름 지정. (기본값: default)

(** label 제거하려면, ""입력)

4) console.count() 예제 - 2번호출한 경우

<script>
 console.count();
 console.count();
</script>

5) console.count() 예제 - label 제거하기

<script>
 console.count("");
 console.count("");
</script>

6) console.count() 예제 - label 지정 후 2번 호출

<script>
 console.count(wmakers);
 console.count(wmakers);
</script>

console.countReset()

1) console.countReset() 예제 - 콘솔창에서 숫자 세기

<script>
 console.count('a')
 console.count('a')
 console.count('a')
 console.count('a')
 console.count('a')
 console.count('a')

 // reset
 console.countReset('a')

 console.count('a')
 console.count('a')
 console.count('a')
</script>

2) console.countReset() 정의

Count로 호출된 횟수를 초기화

- label별로 초기화

3) console.countReset() 구문

console.countReset(label)

label : 선택. 호출 이름 지정. (기본값: default)

(** ""은 제거 불가)

4) console.countReset() 예제

<script>
 console.count('wmakers')
 console.count('')
 console.count('haksoop')
 console.count('')
 console.count('haksoop')

 console.countReset('')

 console.count('wmakers')
 console.count('')
 console.count('')
 console.count('haksoop')
 console.count('wmakers')
 console.count('wmakers')

 console.countReset('wmakers')

 console.count()
 console.count()
 console.count('')
 console.count('wmakers')
 console.count('wmakers')

 console.countReset('haksoop')

 console.count('haksoop')
 console.count()
 console.count()
 console.count('')
 console.count('haksoop')
 console.count('wmakers')
 console.count('wmakers')
</script>
